# vietlott-ai-predictor

![Python](https://img.shields.io/badge/Python-3776AB?style=flat-square&logo=python&logoColor=white)
![PyTorch](https://img.shields.io/badge/PyTorch-EE4C2C?style=flat-square&logo=pytorch&logoColor=white)
![Flask](https://img.shields.io/badge/Flask-000?style=flat-square&logo=flask&logoColor=white)

Machine learning application for analyzing Vietlott lottery patterns using LSTM neural networks, Random Forest, and statistical modeling.

## Features

- LSTM neural network for sequence prediction
- Random Forest ensemble model
- Statistical frequency analysis
- Auto-play engine with configurable strategies
- Live web dashboard for results

## Tech Stack

- **ML**: PyTorch, scikit-learn, NumPy, pandas
- **Backend**: Python Flask
- **Frontend**: Web UI dashboard

## Getting Started

```bash
pip install -r requirements.txt
python backend/app.py
```

## Models

| Model | Method |
|-------|--------|
| LSTM | Recurrent neural network for sequence patterns |
| Random Forest | Ensemble learning on historical features |
| Statistical | Frequency and distribution analysis |

This project is for **educational and research purposes only**. Lottery outcomes are random — no model can reliably predict results. Please gamble responsibly.
